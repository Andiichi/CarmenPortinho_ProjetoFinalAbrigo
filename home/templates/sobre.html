Squad Carmen Portinho

Este projeto foi desenvolvido no âmbito de um bootcamp em Django e Python oferecido pela WoMakersCode. A Squad Carmem Portinho, composta por oito mulheres talentosas e dedicadas, se uniu para criar a plataforma "Lar de Patinhas", um sistema de gestão de adoção responsável de animais.
Construído majoritariamente com Python e Django Web, este projeto incorpora os conhecimentos adquiridos ao longo dos 4,5 meses de curso. O Lar de Patinhas foi idealizado para facilitar o processo de adoção, permitindo que as instituições realizem toda a gestão das adoções, ou seja, a aprovação ou reprovação das pessoas interessadas em adotar um novo amiguinho de quatro patas, promovendo a responsabilidade e o bem-estar dos animais adotados.
